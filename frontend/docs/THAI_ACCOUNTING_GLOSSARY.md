# Thai Accounting Glossary

Use these Thai terms whenever they appear in the UI.

- Contact = ผู้ติดต่อ
- Customer = ลูกค้า
- Vendor = ผู้จำหน่าย
- Supplier = ผู้จำหน่าย
- Partner = คู่ค้า
- Product = สินค้า
- Service = บริการ
- Category = หมวดหมู่
- Payment Channel = ช่องทางการเงิน
- Bank Account = บัญชีธนาคาร
- Cash = เงินสด
- Petty Cash = เงินสดย่อย
- Credit Card = บัตรเครดิต
- Cheque = เช็ค

- Sales = รายรับ
- Expenses = รายจ่าย

- Quotation = ใบเสนอราคา
- Billing Note = ใบวางบิล
- Invoice = ใบแจ้งหนี้
- Tax Invoice = ใบกำกับภาษี
- Receipt = ใบเสร็จรับเงิน
- Receipt/Tax Invoice = ใบเสร็จรับเงิน/ใบกำกับภาษี
- Purchase Order = ใบสั่งซื้อ
- Goods Receipt = ใบรับสินค้า
- Expense Document = เอกสารค่าใช้จ่าย
- Credit Note = ใบลดหนี้
- Debit Note = ใบเพิ่มหนี้

- VAT = ภาษีมูลค่าเพิ่ม
- Input VAT = ภาษีซื้อ
- Output VAT = ภาษีขาย
- Withholding Tax = ภาษีหัก ณ ที่จ่าย

- Chart of Accounts = ผังบัญชี
- Trial Balance = งบทดลอง
- Profit and Loss = งบกำไรขาดทุน
- Statement of Financial Position = งบแสดงฐานะการเงิน
- Cash Flow Statement = งบกระแสเงินสด

- Inventory = สต็อกสินค้า
- Stock Adjustment = ปรับยอดคงคลัง
- Cost of Goods Sold = ต้นทุนขาย

- Payroll = ระบบเงินเดือน
- Employee = พนักงาน
- Department = แผนก
- Position = ตำแหน่ง
- Payslip = สลิปเงินเดือน
- Social Security = ประกันสังคม

Avoid these bad translations:
- Contact = รายชื่อติดต่อ
- Billing Note = บิลเรียกเก็บ
- Credit Note = ใบเครดิต
- Debit Note = ใบเดบิต
- Petty Cash = เงินสดเล็กน้อย
